== Description ==

Adds Negaraku button to your WordPress blog posts - fully customizable.

== Installation ==

1. Upload `negaraku` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Read the instructions from menu Options or Settings >> Negaraku! Options

== Frequently Asked Questions ==

= It`s not working. What do I do? =

Go to Negaraku! Options and click `Update Settings` button.
